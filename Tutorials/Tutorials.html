Tutorials

Welcome to LITESOPH

Starting LITESOPH using the command ( see Usage ) for the first time, you will get into the following Welcome panel:

  • Create LITESOPH Project : Create a new LITESOPH project.

  • Open LITESOPH Project : Open the Predefined Workflow Mode for the further Calculations.

  • About LITESOPH : Click on this button to see “About LITESOPH”.

Modes

In LITESOPH, there are two modes: (1) Predefined Workflow Mode and (2) Task Mode.

  • Predefined Workflow Mode
    • Upload Geometry
    • Charge
    • Multiplicity
    • Select Option
    • Environment
    • Workflow
  • Task Mode
    • Job Type
